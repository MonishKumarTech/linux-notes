# 01 - Linux Fundamentals

## What is Linux?

Linux is an open-source operating system kernel.

A complete Linux operating system is built using:

* Linux kernel
* Shell
* System tools
* Package manager
* Desktop environment
* Applications

Common Linux distributions include Ubuntu, Debian, Fedora, Arch Linux, Linux Mint, and MX Linux.

---

## Why Linux is Used

Linux is widely used because it is:

* Free and open-source
* Stable
* Secure
* Lightweight
* Customizable
* Common in servers and cloud systems
* Useful for system administration and DevOps

Linux is used in:

* Servers
* Cloud platforms
* Networking devices
* Cybersecurity labs
* Development environments
* Embedded systems
* Android-based systems

---

## Kernel

The kernel is the core part of the operating system.

It manages:

* CPU
* Memory
* Storage
* Devices
* Processes
* System resources

Simple recall:

The kernel is the bridge between hardware and software.

---

## Shell

The shell is a program that allows users to interact with the operating system using commands.

Example shells:

* Bash
* Zsh
* Fish

Simple recall:

Shell takes commands from the user and sends them to the operating system.

---

## Terminal

The terminal is the application where we type commands.

Simple recall:

Terminal is the window.

Shell is the command interpreter running inside the terminal.

---

## Distribution

A Linux distribution is a complete operating system built around the Linux kernel.

Examples:

* Ubuntu
* Debian
* Fedora
* Linux Mint
* MX Linux

Simple recall:

Linux kernel + tools + package manager + interface = Linux distribution.

---

## CLI and GUI

## CLI

CLI means Command Line Interface.

In CLI, users type commands to perform tasks.

Example:

```bash
pwd
ls
cd
```

## GUI

GUI means Graphical User Interface.

In GUI, users use windows, icons, buttons, and menus.

Simple recall:

CLI = commands
GUI = mouse and windows

---

## Basic Linux Commands

## pwd

Shows the current working directory.

```bash
pwd
```

Example output:

```bash
/home/demo
```

Recall:

`pwd` means print working directory.

---

## ls

Lists files and directories.

```bash
ls
```

Recall:

`ls` means list.

---

## cd

Changes directory.

```bash
cd Documents
```

Recall:

`cd` means change directory.

---

## whoami

Shows the current logged-in user.

```bash
whoami
```

Example output:

```bash
demo
```

---

## date

Shows the current system date and time.

```bash
date
```

---

## uname

Shows system information.

```bash
uname
```

For more details:

```bash
uname -a
```

## Interview Questions

## 1. What is Linux?

Linux is an open-source operating system kernel used to build Linux-based operating systems called distributions.

## 2. What is a Linux distribution?

A Linux distribution is a complete operating system built using the Linux kernel, system tools, package manager, and applications.

## 3. What is the difference between terminal and shell?

The terminal is the application where commands are typed.
The shell is the program that interprets and runs those commands.

## 4. What is the kernel?

The kernel is the core part of the operating system. It manages hardware, memory, processes, and system resources.

## 5. Why is Linux important for IT support?

Linux is important because many servers, cloud systems, networking tools, and DevOps environments use Linux.
